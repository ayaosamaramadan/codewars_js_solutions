## Convert boolean values to strings 'Yes' or 'No'.

Complete the method that takes a boolean value and return a "Yes" string for true, or a "No" string for false.

---

### Given Code


```js
function boolToWord( bool ){
  //...
}
```

---

### My Solution 


```js
function boolToWord( bool )
{
  return bool ? 'Yes' : 'No';
}
```
