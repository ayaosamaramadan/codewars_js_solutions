## Beginner - Lost Without a Map

Given an array of integers, return a new array with each value doubled.

For example:

[1, 2, 3] --> [2, 4, 6]

---

### Given Code


```js
function maps(x){

}
```

---

### My Solution 


```js
function maps(x)
{
  return x.map(n => n * 2);
}
```
